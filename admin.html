Kontrol Jarak Jauh Nirkabel (Firebase Cloud Sync)

Aktifkan Sinkronisasi Awan (Firebase)

Hubungkan ke cloud untuk mengubah data dari HP/Laptop secara jarak jauh lewat internet.

Cara Menghubungkan Jarak Jauh (Nirkabel):
  1. Buat proyek Firebase gratis di console.firebase.google.com.
  2. Tambahkan aplikasi berbasis Web, lalu salin objek Konfigurasi Firebase (Firebase SDK Config).
  3. Tempelkan nilai konfigurasi tersebut ke kolom input di atas, lalu centang sakelar "Aktifkan Sinkronisasi Awan".
  4. Di konsol Firebase, aktifkan layanan Cloud Firestore di bagian database, dan setel aturan keamanan (Security Rules) menjadi mode baca & tulis (read/write).
  5. Klik tombol "Simpan Perubahan". Aplikasi TV Masjid dan Dashboard di HP Anda sekarang terhubung secara real-time nirkabel!
